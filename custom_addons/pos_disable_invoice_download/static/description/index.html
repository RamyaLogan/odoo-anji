POS Invoice PDF Download

This module allows you to enable or disable the PDF invoice download functionality within the Odoo POS system. It provides flexibility for businesses that do not require PDF invoices for each sale, improving efficiency and streamlining the POS process.

Key Features
  • Enable/Disable PDF Download: Provides the option to toggle the PDF invoice download feature in the POS.
